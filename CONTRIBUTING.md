# Contributing to Abakada Open Source Productivity Tools

Thank you for your interest in contributing to this project. We want to make it as easy as possible for you to join our effort to build a comprehensive glossary of open-source productivity tools.

## How to Contribute

### Adding a New Tool

1.  Navigate to `assets/data/tools.json`.
2.  Add a new entry following the existing JSON structure.
3.  Ensure all fields (name, description, website, license, tags) are accurate.
4.  Submit a Pull Request.

### Reporting Issues

If you find a bug or have a suggestion, please open an issue in the repository. Provide as much detail as possible to help us address the matter.

### Code Guidelines

*   **HTML/CSS/JS**: Maintain the existing coding style and directory structure.
*   **No Emojis**: Please refrain from using emojis in commit messages or code comments to maintain a professional tone.
*   **Accessibility**: Ensure all new UI elements are accessible and support screen readers.
